Astro Stacker 3.1 - uživatelský manuál

Astro Stacker slouží ke skládání astronomických snímků z DSLR, bezzrcadlovek, astronomických kamer a chytrých dalekohledů. Umí načíst sekvenci Light snímků, volitelně použít Flat/Bias/Dark kalibraci, zarovnat snímky podle hvězd nebo komety, složit výsledek a nabídnout základní úpravy náhledu pro kontrolu a export.

Program se spouští s krátkým intro obrázkem a hlavní okno se otevře maximalizované, aby měl uživatel dost prostoru pro náhled, panely i diagnostickou tabulku. Lineární XISF/FIT výstup zůstává vhodný pro další zpracování. Většina úprav v pravém panelu ovlivňuje hlavně náhled a export do PNG/JPG/TIFF, ne původní lineární data.

1. Co program umí

  • Skládání Light snímků ze složky.
  • Podpora PixInsight XISF, FIT/FITS, běžných RAW formátů fotoaparátů a obrazových formátů.
  • Volitelná kalibrace pomocí Flat, Bias a Dark snímků.
  • Zarovnání posunem, ECC afinně, podle hvězd přes RANSAC, perspektivně podle hvězd nebo podle komety.
  • Automatický nebo ruční výběr referenčního snímku.
  • Filtrování nejlepších snímků podle skóre kvality.
  • Diagnostická tabulka kvality snímků.
  • Detekce satelitních stop a maskování pouze zasažených pixelů při skládání.
  • Uložení a načtení profilů nastavení.
  • Náhled s histogramem, křivkami, barvami, vyvážením pozadí a funkcí Balance.
  • Export lineárního XISF/FIT nebo vizuálně upraveného PNG/JPG/TIFF.
  • Ovládání jedné nebo více ASCOM Alpaca kamer se společnou složkou pro Live Stack.
  • Volitelný PixInsight wrapper pro spuštění CLI stackeru přímo z PixInsightu.

2. Rychlý postup pro běžné skládání

  1. Klikni na Choose folder / Vybrat složku a vyber složku s Light snímky.
  2. Pokud jsou ve stejné složce i JPG/PNG/BMP/TIFF náhledy, zapni RAW only / Pouze RAW.
  3. Pro běžnou deep-sky sekvenci nech nastaveno:
  • Alignment / Zarovnání: Star alignment - stars + RANSAC
  • Stacking / Skládání: Median / Medián
  • Auto reference / Automaticky vybrat nejlepší referenci: zapnuto
  • Use only best frames / Použít jen nejlepší snímky: podle potřeby zapnuto
  1. Klikni na Start stacking / Spustit skládání.
  2. Po dokončení zkontroluj náhled, diagnostickou tabulku a hlášení o počtu použitých/vyřazených snímků.
  3. Výsledek ulož přes File > Save result as / Soubor > Uložit výsledek jako.

2A. ASCOM Alpaca kamery a vícekamerový Live Stack

Otevři Camera > ASCOM Alpaca cameras.... Stejné okno slouží pro jednu kameru i pro synchronizovanou skupinu kamer.

  1. Zadej adresu a port Alpaca serveru. Výchozí nastavení je 10.42.42.1:80.
  2. Kliknutím na Refresh vyhledej dostupné kamery.
  3. Ve sloupci Use označ všechny kamery, které se mají účastnit sekvence.
  4. Klikni na Connect selected.
  5. Nastav expozici, počet snímků, interval, gain, binning, chlazení a cílovou teplotu.
  6. Vyber jednu společnou výstupní složku. Podle potřeby zapni automatické spuštění Live Stacku.
  7. Klikni na Start synchronized sequence.

Sloupec Camera/ID file spojuje název kamery s jedinečným identifikátorem souborů. Identifikátor se ukládá do názvu vytvořeného souboru a do FIT položky CAMID, takže lze rozlišit snímky i z několika stejných kamer.

Každá označená kamera pracuje samostatně, zobrazuje vlastní stav a ukládá dokončené expozice do stejné složky. Live Stack čeká na úplné dopsání souboru a umí bezpečně zpracovat snímky, které dorazí z více kamer téměř současně.

Všechny kamery musí být z počítače dostupné současně, obvykle přes jednu LAN, Wi-Fi router nebo více správně nastavených síťových adaptérů. Přímé Wi-Fi spojení s kamerou může odpojit běžný internet. Vyhledávání používá Alpaca discovery i přímý dotaz na zadanou adresu.

Astro Stacker podporuje obrazová data Alpaca JSON i binární odpověď ImageBytes. Bayer metadata dodaná kamerou se zachovají ve FIT souboru pro automatický debayering. Rozhraní bylo otestováno s kamerami Pegasus SmartEye a Seestar.

Nápověda a aktualizace

Astro Stacker krátce po spuštění zkontroluje GitHub Releases, zda není dostupná novější verze. Pokud nová verze existuje, zobrazí upozornění a může otevřít stránku ke stažení. Kontrolu lze spustit i ručně přes Nápověda > Zkontrolovat aktualizace.

Výběr složky a náhledu

Choose folder / Vybrat složku načte pracovní složku se snímky. Program v seznamu ukazuje Light, Flat, Bias a Dark snímky. Po stackování se v seznamu označí:

  • * referenční snímek,
  • x vyřazený snímek.

Rozbalovací seznam slouží také k rychlé kontrole jednotlivých snímků.

RAW only / Pouze RAW

Tato volba při stackování ponechá XISF, FIT/FITS a foto RAW soubory, například ARW, CR2, CR3 apod. Vynechá JPG, PNG, BMP a TIFF. Hodí se, když chytrý dalekohled nebo fotoaparát ukládá do složky současně pracovní snímky i náhledy.

XISF soubory z PixInsightu lze použít jako Light i kalibrační snímky. Podporovány jsou také komprimované XISF soubory. XISF bez explicitních CFA/Bayer metadat se považuje za monochromatický; CFA XISF se debayeruje automaticky nebo podle ručně zvolené Bayer masky.

U Bayer XISF/FIT/FITS sekvencí se kalibrace aplikuje na senzorovou mozaiku před debayeringem. Hlavičky velkých FITů se kontrolují bez načítání celého obrazu a vlastní načtení, debayering a alignment běží při zapnutém multiprocessingu v paralelních pracovních procesech.

Zarovnání

Translation / Pouze posun

Rychlé zarovnání pouze posunem. Vhodné pro malé drifty bez rotace.

Calibration/no alignment / Kalibrační snímky - bez zarovnání

Používá se pro skládání kalibračních snímků pixel na pixel. Bayer FIT/XISF/RAW snímky zůstanou jako původní 2D senzorová mozaika a nedebayerují se. Tento režim je správný pro vytvoření Master Bias, Master Dark nebo Master Flat FIT souborů. Náhled může 2D master zobrazit jako neutrální monochromatické RGB pouze kvůli obrazovce; uložený FIT zůstane mono/mozaika.

ECC affine / Afinní ECC

Zarovnání posunem, rotací a změnou měřítka pomocí obrazové korelace. Může být pomalejší a u slabých objektů nemusí být ideální.

Star alignment + RANSAC

Hlavní doporučený režim. Program detekuje hvězdy, páruje je s referenčním snímkem a přes RANSAC odhadne transformaci. Snímky bez platného zarovnání vyřadí.

Comet alignment

Skládá na kometu podle označené polohy komety.

Star + Comet

Vytvoří samostatný výstup pro hvězdy a kometu.

Automatická a ruční reference

Automatically choose best reference / Automaticky vybrat nejlepší referenci vybere snímek s nejlepším skóre kvality. Skóre vychází hlavně z ostrosti a počtu detekovaných hvězd.

Pokud automatická reference u některé sekvence nefunguje ideálně, vyber v seznamu vhodný snímek a klikni na Use current frame as reference / Použít aktuální snímek jako referenci. Tím se automatická reference vypne a program použije zvolený snímek.

Quality filter / Použít jen nejlepší snímky

Program spočítá skóre kvality pro každý Light snímek. Skóre kombinuje:

  • ostrost snímku,
  • počet detekovaných hvězd.

Volba Keep / Ponechat určuje, kolik procent nejlepších snímků se použije. Například 80 % znamená, že nejhorších 20 % bude vyřazeno ještě před alignmentem.

Důležité: snímek může projít filtrem kvality, ale později být vyřazen, pokud selže zarovnání.

Max. star drift / Max. drift hvězd

Určuje největší očekávaný posun hvězd vůči referenci. U běžných sekvencí stačí menší hodnota. U EAA, ditheringu nebo chytrých dalekohledů může být potřeba hodnotu zvýšit.

Ignore border / Ignorovat okraj

Program při hledání hvězd ignoruje okraje snímku. To pomáhá, pokud jsou na okraji větve, šum, vinětace, černé rohy po rotaci nebo jiné rušivé struktury.

Strict star filter / Přísný filtr hvězd

Přísnější kontrola tvaru hvězd. Pomáhá proti větvím, stopám a protáhlým artefaktům. Pokud program odmítá příliš mnoho použitelných snímků, může někdy pomoci tuto volbu vypnout.

Satellite trail / Satelitní stopa

Detekuje dlouhé rovné stopy družic nebo letadel a podezřelé snímky označí v tabulce Frame quality. Při skládání Astro Stacker prodlouží masku po celé nalezené přímce od okraje k okraji. Vynechají se pouze pixely stopy; nepoškozená část expozice zůstane využita.

Odmítání snímků v Live Stacku

Live Stack používá průběžné sigma odmítání pixelů pro hotpixely, kosmické záření, letadla a satelitní stopy. Volba Odmítat protažené hvězdy může odmítnout celý Live Stack snímek, pokud jsou změřené hvězdy příliš protáhlé. Pomáhá to při větru, chybách trackingu nebo vibracích montáže.

4. Diagnostická tabulka snímků

Ve spodní části je tabulka Frame quality / Vyhodnocení snímků. Slouží ke kontrole, proč byly snímky použity nebo vyřazeny.

Sloupce:

  • # pořadí snímku,
  • File / Soubor název souboru,
  • Score / Skóre celkové skóre kvality,
  • Stars / Hvězdy počet detekovaných hvězd,
  • Sharpness / Ostrost ostrost podle Laplacian variance,
  • Status / Stav použití snímku,
  • Ref označení reference.

Typické stavy:

  • Used / Použit - snímek byl složen.
  • Reference - referenční snímek.
  • Excluded by quality / Vyřazen kvalitou - snímek neprošel filtrem kvality.
  • Rejected alignment / Selhal alignment - snímek byl kvalitní dost, ale nepodařilo se ho zarovnat.
  • Skipped / Vynechán - snímek nebyl součástí stacku, například kalibrační nebo nepoužitý typ.

Kliknutí na řádek tabulky přepne náhled na daný snímek.

Kontrola snímků před skládáním

Volba Review frames before stacking / Zkontrolovat snímky před skládáním vloží před samotné skládání ruční kontrolní krok.

Postup:

  1. Zapni Zkontrolovat snímky před skládáním.
  2. Klikni na Spustit skládání.
  3. Program pouze spočítá kvalitu snímků, vybere referenci a naplní tabulku Frame Quality.
  4. V tabulce vyber snímek, který nechceš použít.
  5. Stiskni mezerník. Stav se změní na Vyřazený a řádek zčervená.
  6. Dalším stiskem mezerníku lze snímek znovu povolit.
  7. Klikni na Pokračovat ve skládání.

Referenční snímek nelze mezerníkem vyřadit. Pokud chceš vyřadit referenci, nejprve vyber jiný referenční snímek.

Ruční vyřazení má přednost před automatickým výběrem. Automatický quality filter stále určí první výběr, ale před finálním alignmentem můžeš ručně odebrat další snímky.

5. Kalibrace Flat, Bias a Dark

Program umí použít kalibrační snímky dvěma způsoby:

  • automaticky, pokud jsou ve vhodně pojmenovaných složkách,
  • ručně přes tlačítka Flat, Bias a Dark v pravém panelu.

Kalibrace se aplikuje před zarovnáním a stackováním Light snímků.

Pokud chceš vytvořit master kalibrační soubor, vyber Kalibrační snímky - bez zarovnání. V tomto režimu Astro Stacker nepoužije Flat/Bias/Dark z pravého panelu, nespouští quality filter ani automatickou referenci a nedebayeruje Bayer data. Výsledek lze uložit jako lineární Master FIT.

Tlačítko Reset calibration / Reset kalibrace odstraní vybrané kalibrační snímky.

6. Skládání komety

Pro kometu je nejspolehlivější označit její polohu ve dvou snímcích.

  1. Vyber složku se sekvencí.
  2. Klikni na Comet First / Kometa první.
  3. Program načte první snímek. Klikni v náhledu na jádro komety.
  4. Klikni na Comet Last / Kometa poslední.
  5. Program načte poslední snímek. Klikni na jádro komety.
  6. Program zná pohyb komety mezi začátkem a koncem sekvence.
  7. Vyber režim Comet alignment nebo Star + Comet.
  8. Spusť skládání.

Comet Clear / Kometa smaž smaže uloženou první i poslední polohu komety.

Volby pro kometu:

  • Max. comet motion / Max. pohyb komety - největší očekávaný pohyb komety.
  • Refine comet position / Jemně doladit kometu - dohledá jádro lokální korelací.
  • Comet template / Šablona komety - velikost šablony kolem jádra.
  • Comet search / Hledání komety - jak daleko od předpokládané polohy se smí jádro dohledat.

7. Předvolby a profily nastavení

Předvolby se ukládají jako JSON profil.

Použij:

  • File > Save settings profile / Soubor > Uložit profil nastavení
  • File > Load settings profile / Soubor > Načíst profil nastavení

Po výběru pracovní složky přes Choose folder / Vybrat složku se dialog pro uložení výsledku i profilu nastavení otevře přímo v této složce.

Profil ukládá nastavení stackování, kalibrace, komety i pravého panelu. Hodí se například pro různé typy dat:

  • DSLR RAW,
  • Seestar/Vespera EAA,
  • komety,
  • kalibrační snímky,
  • čistý lineární FIT workflow.

8. Pravý panel - náhled a úpravy

Pravý panel slouží ke kontrole a vizuálním úpravám výsledku. Ovladače jsou rozdělené do záložek pro barvy, korekce a AI nástroje. Tyto úpravy jsou určené hlavně pro náhled a export do PNG/JPG/TIFF. Lineární XISF/FIT výstup zůstává vhodný pro další zpracování.

Po dokončení se stack zobrazí lineárně, takže může působit téměř černě. Tlačítko Balance provede stretch pouze pro náhled; lineární data nezmění.

Malou šipkou v levém horním rohu lze pravý panel dočasně schovat. Kliknutím na

zbývající úzký proužek se šipkou se panel znovu zobrazí.

Show stacked image / Zobraz složený obraz

Vrátí náhled na původní složený obraz bez ořezu, neutralizace a vizuálních úprav.

Balance

Tlačítko Balance automaticky nastaví rozumný náhled lineárního snímku:

  • neutralizuje pozadí pro náhled,
  • nastaví black point,
  • nastaví gamma,
  • snaží se zobrazit slabý lineární obraz čitelněji.

Balance nemění lineární FIT data. Je to rychlá pomůcka pro náhled.

Black point, White point, Gamma

Základní křivky pro zobrazení:

  • Black point posouvá černý bod.
  • White point určuje světlý bod.
  • Gamma mění střední tóny.

Black point a White point lze nastavovat jemně po 0,05, například na hodnotu 2,25.

STF strength / Síla STF

Nastavuje intenzitu Screen Transfer Function pouze pro náhled a vizuální PNG/JPG/TIFF export. Zobrazená stupnice roste běžně zleva doprava: vlevo je 0 a vpravo 10. Účinek je záměrně obrácený: 0 znamená nejsilnější STF pro slabé struktury, 5 standardní náhled a 10 nejjemnější STF. Lineární XISF/FIT export se nemění.

Classic a AI DRUNet odšumění

Classic používá vestavěný CPU algoritmus. AI DRUNet používá neuronový model ONNX a obvykle poskytuje výraznější potlačení jasového i barevného šumu.

  • models/drunet_color.onnx je výchozí model pro barevné RGB snímky.
  • models/drunet_gray.onnx je určený pro monochromatické snímky.
  • Po prvním výpočtu Astro Stacker uloží adaptivní šumovou mapu. Gamma, STF, kontrast, barvy, Balance, odstranění gradientu a další ovladače ji znovu použijí bez opakovaného spuštění neuronové sítě.
  • Nový výpočet se spustí při změně síly odšumění, modelu, zdrojového snímku nebo ořezu.

ONNX Runtime na macOS přednostně používá CoreML, ve Windows CUDA nebo DirectML. Pokud GPU akcelerace není dostupná, automaticky se použije CPU. Výpočet je pomalejší, ale funkce zůstává dostupná.

AI dekonvoluce hvězd

AI dekonvoluce hvězd doostří a jemně zmenší kompaktní hvězdy pomocí přiloženého stellar ONNX modelu Cosmic Clarity. Ovladač Velikost hvězd mění poloměr masky omezené pouze na hvězdy, aby se široké struktury galaxií a mlhovin nezpracovávaly jako hvězdy.

  • models/cosmic_clarity_stellar.onnx se vybere automaticky.
  • První výpočet se uloží do cache. Síla, Velikost hvězd, STF, gamma, kontrast a barvy znovu použijí již vypočtený výsledek modelu.
  • Na macOS se přednostně používá CoreML, ve Windows CUDA nebo DirectML, pokud jsou dostupné.
  • Efekt se použije pouze pro náhled a vizuální PNG/JPG/TIFF export. Lineární XISF/FIT výstup zůstane beze změny.

Vignette removal / Odstranění vinětace

Jemně zesvětlí okraje a rohy. Není to náhrada skutečného Flat snímku, ale může pomoci pro rychlý vizuální náhled.

Synthetic flat / Umělý flat

Odhadne hladké pozadí ze složeného obrazu a použije ho jako jemnou korekci. Používej opatrně, zvlášť u rozsáhlých mlhovin, kde může program část mlhoviny považovat za pozadí.

Color background correction / Korekce barevného pozadí

Potlačuje hladký barevný závoj pozadí po RGB kanálech. Je užitečná hlavně u velmi barevně nevyvážených snímků, například růžové nebo fialové pozadí z chytrých dalekohledů.

SCNR Green

Šetrně potlačuje pouze zelený přebytek metodou Maximum Neutral. Zelený kanál se přibližuje silnějšímu z červeného a modrého kanálu, takže se zachová více skutečného signálu zelenomodrých mlhovin než při stažení zelené k průměru R/B.

Contrast, Saturation, RGB

  • Contrast / Kontrast mění kontrast náhledu.
  • Saturation / Saturace mění barevnost.
  • Red, Green, Blue / Červená, Zelená, Modrá ručně upravují jednotlivé kanály.

Histogram

Histogram ukazuje jasový kanál a RGB kanály. Pomáhá kontrolovat, zda nejsou data příliš oříznutá v černé nebo bílé.

Turn angle / Úhel natočení

Pootočí aktuální snímek o nastavený úhel. Hodí se tehdy, když je složený obraz mírně šikmo a chceš jej před finálním ořezem srovnat.

Tlačítko Výběr / Select zapne ruční ořez myší. V náhledu natáhni obdélník;

vybraná oblast zůstane a vše mimo ni se odstraní. Hodí se, když je potřeba

oříznout jen jednu stranu nebo přesně vybrat oblast bez špatných okrajů. Ruční

ořez pracuje s aktuálním obrazem, případně už pootočeným pomocí Turn angle.

Undo / Zpět

Tlačítko Undo vrátí předchozí stav vizuálních úprav náhledu. Hodí se po akcích jako Balance, Auto WB, odšumění, Turn angle nebo ruční ořez.

Auto White Balance

Pokusí se automaticky vyvážit bílé/šedé oblasti. U některých astrofotografií může být vhodnější ruční úprava RGB nebo korekce barevného pozadí.

Neutralize background / Neutralizovat pozadí

Snaží se srovnat barvu pozadí. Pozadí vybírá podle geometrie, jasu a absence hvězd, nikoli podle podobné síly RGB kanálů. Zvládne proto lépe také snímky s velmi slabou modrou nebo jinou barevnou složkou. Používá aditivní posuny, aby slabý kanál zbytečně nezesiloval svůj šum. Nejlépe funguje, když je v obraze dost skutečného pozadí bez rozsáhlé mlhoviny.

Flip horizontal / vertical

Překlopí náhled horizontálně nebo vertikálně.

Fit a 1:1

  • Fit přizpůsobí snímek oknu.
  • 1:1 zobrazí skutečné pixely.

9. PixInsight wrapper

Release balíček obsahuje volitelný AS_Stacker wrapper pro PixInsight. Wrapper umožňuje spustit CLI verzi stackeru přímo z PixInsightu a po dokončení otevřít výsledný FIT v PixInsightu.

Pro wrapper je potřeba nastavit Python prostředí, ve kterém jsou nainstalované potřebné balíčky.

Balíček wrapperu ve verzi 3.1 obsahuje aktuální skládací jádro, volbu Satellite trail, oba DRUNet modely a stellar model pro AI dekonvoluci hvězd. Celou složku AS_Stacker_PI_Plugin ponech pohromadě. CLI wrapper vytváří lineární FIT výstup, proto se na složený FIT nepoužije odšumění ani dekonvoluce určená pouze pro náhled.

10. Doporučené postupy

Běžná deep-sky sekvence

  • Star alignment + RANSAC
  • Median / Medián
  • Auto reference zapnuto
  • Quality filter 80-100 %
  • Strict star filter zapnuto

Sekvence s velkým driftem nebo ditheringem

  • Zvyš Max. star drift.
  • Zkus ruční referenci ze středu sekvence.
  • Sleduj diagnostickou tabulku, jestli snímky padají na kvalitě nebo alignmentu.

Chytré dalekohledy a EAA

  • Zapni RAW only, pokud jsou ve složce i náhledové obrázky.
  • U rotace a tmavých rohů použij ořez.
  • U růžového/fialového pozadí použij Color background correction.

Komety

  • Označ první i poslední polohu komety.
  • Použij Comet alignment nebo Star + Comet.
  • U slabého jádra nech zapnuté jemné doladění komety.

11. Co dělat při problémech

Program vyřazuje mnoho snímků

Zkontroluj tabulku Frame quality:

  • pokud jsou snímky Excluded by quality, sniž filtr kvality nebo zvyš Keep %;
  • pokud jsou Rejected alignment, zvyš Max. star drift, změň referenci nebo zkus jiný režim zarovnání.

Snímek je barevně divný

Zkus postupně:

  1. Balance,
  2. Color background correction,
  3. Neutralize background,
  4. ruční RGB posuny,
  5. případně ořez okrajů a neutralizaci znovu.

Výsledek má tmavé rohy

Je to běžné u rotace, ditheringu nebo velkého driftu. Použij Turn angle, ruční výběr cropu nebo výsledek dále ořízni v externím editoru.

FIT výstup vypadá tmavě

To je normální. FIT je lineární a potřebuje stretch. Použij Balance pro náhled nebo výstup zpracuj v PixInsightu, Sirilu či jiném astrofoto editoru.

12. Instalace

Windows instalátor

Podpora ASCOM Alpaca kamer používá standardní síťové komponenty a nevyžaduje žádný další Python balíček.

  1. Spusť AstroStacker31_Setup.exe.
  2. Instalátor obsahuje CPU aplikaci a volitelně NVIDIA CUDA build. CUDA podporu zvol jen na Windows počítači s podporovanou NVIDIA kartou a aktuálním NVIDIA ovladačem.
  3. Instalátor obsahuje také Microsoft Visual C++ Runtime a podle potřeby jej spustí tiše na pozadí.
  4. Pro uživatele, kteří chtějí co nejmenší CPU verzi bez AI odšumění, CUDA, ONNX Runtime, PyTorch a PixInsight wrapperu, lze vytvořit AstroStacker31_Lite_Setup.exe.

macOS DMG

  1. Otevři AstroStacker31_macOS.dmg.
  2. Přetáhni AstroStacker.app do Applications.
  3. Balík aplikace obsahuje ONNX Runtime pro AI nástroje a PyTorch pro Apple Metal/MPS akceleraci skládání.

Zdrojová instalace pro testování

Vývojář může na macOS stále použít install_astro_stacker_macos.command, případně ve Windows vytvořit Python prostředí ručně. Běžní uživatelé by měli použít samostatný instalátor nebo DMG.

Samostatný Windows instalátor a macOS DMG už obsahují intro obrázek, český/anglický manuál, oba DRUNet modely, stellar model pro AI dekonvoluci hvězd a balíček PixInsight wrapperu. Nevyžadují samostatnou instalaci Pythonu.
